2048

2048 is an addictive number puzzle game that tests your strategic thinking. Slide numbered tiles on a grid to combine matching numbers and create a tile with the number 2048. With simple rules but challenging gameplay, this game offers endless entertainment while exercising your brain. Are you up for the challenge of reaching 2048 or beyond?

4.8 (2,345 ratings)

How to Play

  1. Use arrow keys or swipe to move all tiles in one direction
  2. When two tiles with the same number touch, they merge into one tile with double the value
  3. After each move, a new tile appears (either 2 or 4)
  4. Plan your moves carefully to create space for new tiles
  5. The game ends when the board is full and no more moves are possible

Pro Tips

  • Focus on keeping your highest number tile in one corner
  • Create a chain of decreasing numbers from your highest tile
  • Avoid moving tiles in the direction of your highest value corner if possible
  • Think several moves ahead - like in chess
  • When the board gets crowded, prioritize clearing space over merging small values
